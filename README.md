# NanoML
A Tensor based Nano Machine Learning Library built in C++.
<br/><br/>
Only supports upto 2D tensor as of now.
<br/><br/>
Only supports addition and multiplication(scalar,dot,matrix),transpose as of now.
<br/>
Support of other operators and functions will come later.
<br/><br/>
Minimal Version. Initial launch. No backprop now, later will be added.
